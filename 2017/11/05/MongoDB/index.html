MongoDB

发表于 2017-11-05

简介

MongoDB是一个基于分布式文件存储的数据库。由C++语言编写。旨在为WEB应用提供可扩 展的高性能数据存储解决方案。MongoDB是一个介于关系数据库和非关系数据库之间的产 品，是非关系数据库当中功能最丰富，最像关系数据库的。它支持的数据结构非常松散，是类 似json的bson格式，因此可以存储比较复杂的数据类型。Mongo最大的特点是它支持的查询 语言非常强大，其语法有点类似于面向向对象的查询语言，几乎可以实现类似关系数据库单表查 询的绝大部分功能，并且还支持对数据建立索引。

MongoDB特点

面向集合存储，支持复制和故障恢复，使用高效的二进制存储，包括大型对象

思想

MongoDB所谓的面向集合，说到底就是数据被分组存储在数据集中，也就是集合 一个集合相当于关系型数据库的一张表 一个行就是一篇文档 列就是关系型数据库的大多的属性比如“name” 主键本身就有 并且还没有外键

安装

{

[root@CentOS ~]# tar -zxvf mongodb-org-3.6.repo1.gz
[root@CentOS ~]# mv mongodb-org-3.6.repo1.gz mongoDB
[root@CentOS ~]# vim /etc/yum.repos.d/mongodb-org-3.6.repo1
[root@CentOS ~]# mkdir data
[root@CentOS mongoDB]#
[root@CentOS bin]# ./mongod --port 27017 --dbpath /root/data
默认本机连接别的端口[root@CentOS  bin]# mongod --port 27017 --dbpath /root/data/db --bind_ip 192.168.204.132
[root@CentOS mongoDB]# ./mongo --port 27017
即连接成功

}

常用MongoDB操作命令和使用

创建普通的集合也就是类似数据库表
db.createCollection(“t_user”)
添加表中的属性
db.t_user.insert({name:”caotian”})
db.t_user.insert({name:”zhouxu”},{name:”chenglong”})

删除表
db.t_user.drop()
删除表中的数据
db.t_user.remove({name:”caotian”})
查看表
db.t_user.find()
按条件查询db.t_user.find(name:”zhouxu”)
